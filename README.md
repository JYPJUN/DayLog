## 3. 기획 의도 및 목표
연인들은 영상통화를 통해 일상을 공유하고 재미있는 대화를 나누며 추억을 쌓습니다.   
DayLog는 연인들이 영상통화 중 나눈 재미있고 중요한 대화의 하이라이트를 AI를 통해 클립으로 생성하고, 커플의 추억을 관리하는 앱입니다.
